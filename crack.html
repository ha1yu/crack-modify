crack-modify 命令配置 v1.3.0

常见服务弱口令爆破 · 勾选参数自动生成命令，复制到终端执行

格式：ip:port / CIDR(≤/24) / IP段(1-128) / 逗号列表。协议由 -m 决定, 端口任意
▾ -i / -f 目标格式说明（点击展开）

基本格式：IP规范:端口

IP 部分可以是单个 IP / CIDR / IP段 / 逗号列表；端口必填（任意端口均可）。协议由 -m 指定，与端口无关。

IP 部分写法

写法含义例子
单个 IP一个目标127.0.0.1
CIDR网段展开（≤ /24，防误爆）192.168.1.0/24
IP 段末位数字范围10.0.0.1-128
逗号列表多个 IP10.0.0.1,10.0.0.5

完整例子

-i 的值展开为
127.0.0.1:33061 个目标（-m mysql 时当 mysql 爆破）
127.0.0.1:33071 个目标（-m mysql 时, 非标准端口也当 mysql）
192.168.1.0/24:445256 个目标（-m smb 时全当 smb）
10.0.0.1-128:22128 个目标（-m ssh 时全当 ssh）
10.0.0.1,10.0.0.5:33062 个目标（逗号列表, -m mysql）

注意事项

• 必须带端口：192.168.1.0/24（无端口）会被跳过，要写 192.168.1.0/24:445。
• -m 决定协议：端口与服务标准端口不同也没关系（如 mysql 跑在 3307，只要 -m mysql）。
• -f 目标文件：每行一个目标，格式同上。
0 关闭限速；全局令牌桶，真实按此间隔发请求
